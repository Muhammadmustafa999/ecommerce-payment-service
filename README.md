# E-Commerce Payment Service

A microservice handling payment processing for our e-commerce platform.

## Tech Stack
- Django 3.2 (REST API)
- Stripe Payment Integration  
- Redis for task queuing
- PostgreSQL database
- 

## Security
This service handles PCI-DSS sensitive payment data.
CI/CD security scanning is powered by CI/CD ThreatIntel.

## Setup
pip install -r requirements.txt
python manage.py runserver
